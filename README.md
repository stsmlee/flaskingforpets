## Flasking for Pets
<p align = "center"><img width="500" alt="Screenshot Home Screen" src="https://user-images.githubusercontent.com/84687089/212180345-47c0a320-3b9c-4b18-ba1c-631a5c6fb91c.png"></p>
 
This Flask web application uses Petfinder’s RESTful API to search for and return results for adoptable pets, using the user’s selected filters as parameters. 

<p align = "center"><img width="500" alt="Screenshot Cat Search" src="https://user-images.githubusercontent.com/84687089/212180359-e34f93ff-c32f-487d-bcec-7ec13d417e4b.png"> <img width="500" alt="Screenshot Results" src="https://user-images.githubusercontent.com/84687089/212189434-d39b1264-2188-45eb-a667-295a5ecdbbc2.png"></p>

Users can register and log in to save their searches, and then with one click (What’s News?), the app will check and notify the user whether or not there are new results for each saved search. 

<p align = "center"><img height="100" alt="Screenshot Whats News Waiting Image" src="https://user-images.githubusercontent.com/84687089/212178536-04a1e014-a65e-42bc-9dc3-f990cf53fc87.png"> <img height="100" alt="Screenshot Whats News Results" src="https://user-images.githubusercontent.com/84687089/212178537-9c1fc5c4-b4fb-4002-8a4d-b64f93e2eac2.png"></p>

Registered users can manage their account, which includes deleting saved searches, changing nickname and/or password, and deleting their account altogether.

<p align = "center"><img width="500" alt="Screenshot Manage Account Page" src="https://user-images.githubusercontent.com/84687089/212179988-29116c50-828e-4da0-a75f-933c7b9d1ea9.png"></p>

## Squeerdle
<p align = "center"><img width="500" alt="Screenshot Squeerdle Win" src="https://user-images.githubusercontent.com/84687089/212180971-72a6aeba-e737-4ce3-8404-c6b717fb2d77.png"></p>

Registered users can also play the bonus minigame of Squeerdle. It is much like NYT’s Wordle, except users can:

  * Get a new random puzzle from the database on demand (instead of only one daily puzzle).
  
  * Keep track of incomplete puzzles and view completed puzzles.
<p align = "center"><img width="500" alt="Screenshot Squeerdle Incomplete" src="https://user-images.githubusercontent.com/84687089/212183825-7f72e883-9051-4d63-927e-86fbdfec86e1.png"><br />
<img width="500" alt="Screenshot Squeerdle Complete" src="https://user-images.githubusercontent.com/84687089/212183975-c934310e-31d7-46fa-8673-0081fb83fbea.png"></p>

  * Add new words to the database and send these words to others with known usernames.

<p align = "center"> <img width="500" alt="Screenshot Squeerdle Create" src="https://user-images.githubusercontent.com/84687089/212183996-37ece2f1-695f-4a14-b39f-27fb54037de4.png"><br />
<img width="500" alt="Screenshot Squeerdle Inbox" src="https://user-images.githubusercontent.com/84687089/212184076-03d7cbef-166a-4f2c-b4a9-15f5219e9ac0.png"></p>
  
  * Keep track of their created puzzles’ play and win counts.
